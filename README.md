## Тестирование домашней страницы effective-mobile

### Будет протестирована главная страница сайта effective-mobile, а также основные функциональности, переходы по разделам.
### Автоматизированное тестирование:
* тестирование перехода по разделам главной страницы
* Раздел О нас: кликабельность, ожидаемый переход по странице, ожидаемый текст отображения после клика на кнопку раздела
* Раздел Услуги: кликабельность, соответствующее отображение текста информации
* Раздел Отзывы: после клика отображение соответствующих отзывов о компании
* Раздел Контакты: после нажатия на кнопку Контакты отображение инпутов для заполнения инфорации и заказа обратного звонка
